When submitting multiple proteins to ProLinks, either for getting their functional linkages or to find the set of linkages shared by all of them, the protein identifiers must be separated by commas, spaces or any combination of these. Here are a few examples using Locus Tags:

  • Rv0753c Rv1187,Rv2858c Rv0234c Rv1731, Rv3293

  • Rv0753c, Rv1187
    Rv2858c
    Rv0234c,
    Rv1731 Rv3293

  • Rv0753c
    Rv1187,
    Rv2858c
    Rv0234c
    Rv1731,
    Rv3293
If you are submitting a file, make sure it is a plain text file using ASCII characters only.
