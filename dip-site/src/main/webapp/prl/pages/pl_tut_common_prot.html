Find functional linkages shared by multiple proteins

This tutorial is divided in 2 sections:
  • The input form: General description of the input form.

  • Submit your query: Query ProLinks by entering a list of protein IDs or uploading a file with the IDs

The input form

The input form to search for linkages shared between several proteins is shown in Figure 1a. As you can see, the structure of the form is identical to that shown in the tutoral to find linkages for multiple proteins. Similarly, the user must enter a valid e-mail address (results are returned by e-mail only), and either copy/paste a list of protein identifiers or upload a text file with the identifiers. The search parameters tab offers exactly the same parameters as the input form to submit multiple proteins.

IMPORTANT: if both a list of IDs and a file with IDs are provided, the entered list of IDs takes precedence and the file is ignored.


Figure 1. Input form. The user must provide a valid e-mail address, and either enter the list of IDs or upload a file with the IDs. By clicking on any question mark, a window will open with an explanation of the corresponding parameter. (click on the figure to see the full size image).


Submit your query

If you click on the link: (see valid formats) a window will open with an explanation of the accepted formats for the list you submit. The window shows 6 Locus tags in three different formats as an example. Enter the same list or select the same file that you used in the previous tutorial.

When you click on the Search ProLinks button, ProLinks will display a page indicating that your query has been submitted to the cluster. Depending on the number of IDs you submit it will take longer for ProLinks to process your query. When it is done, you will get an e-mail with a link indicating where you can see or download the text file with your results. This file includes a header with an explication of each column. Figure 2 shows the results file if you submitted the same IDs as those used in the previous tutorial.

Figure 2. Results file with shared links among multiple proteins. The file includes a header with a description of each column. Only one common link was found at Depth of search 1. Increase the depth to 2 or 3 if you want to find more shared links.
