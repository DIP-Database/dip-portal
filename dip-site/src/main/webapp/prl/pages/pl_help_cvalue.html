Given that the individual scores reported by each inference method in ProLinks were calculated under different assumptions and methodologies, these values are not directly comparable. The confidence value is a way to normalize these scores to make them directly comparable and easy to interpret.

Briefly, this value is the global probability that an inferred functional linkage is a true positive given the scores reported by each individual inference method. The closer to one, the more likely the inferred linkages will be true positives.

To keep the number of inferences for your query reasonable, ProLinks only accepts values ≥0.4. Please, contact us (e-mail: prolinks@mbi.ucla.edu) if you would like to search inferences at lower levels of confidence.
