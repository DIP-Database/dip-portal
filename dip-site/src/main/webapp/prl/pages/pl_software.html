Software
Prolinks was developed with:
MiSink Plugin - This Cytoscape plugin allows for easy, interactive import of DIP/ProLinks interactions into the Cytoscape network visualization platform. Cytoscape - software platform for visualizing molecular interaction networks and integrating these interactions with gene expression profiles and other types of data.
Solr - the popular, blazing fast open source enterprise search platform. Solr is highly scalable, providing distributed search, index replication, and it powers the search and navigation features of many of the world's largest internet sites. MySQL - open source database software. MySQL is a key part of LAMP (Linux, Apache, MySQL, PHP/Perl/Python), the fast-growing open source enterprise software stack. More and more companies are using LAMP as an alternative to expensive proprietary software because of its lower cost and freedom from platform lock-in.
PostgreSQL (Postgres) - open source, object-relational database management system. Spring (2.5) - open source application framework for the Java platform.
YUI 2 Library - JavaScript and CSS web browser user interface library. Apache Struts (2.x) - web application framework for developing Java EE web applications.
Apache Maven - build manager for Java projects. Subversion - open source version control system.
PHP - a widely-used open source general-purpose scripting language that is especially suited for Web development and can be embedded into HTML. Perl - highly capable, feature-rich programming language with over 23 years of development. Perl runs on over 100 platforms and is suitable for both rapid prototyping and large scale development projects.
