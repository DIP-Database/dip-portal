ProLinks Help

How to make a query

There are five types of queries you can submit to ProLinks:

  • Submit keywords associated with a protein: Keywords can be protein identifiers (click here for a list of supported IDs), gene names, protein names, genome names, words or phrases describing function, etc. If your query identify only one protein, it is directly used to search for functional linkages. If your query returns multiple proteins, you will be presented with a list of matches and you will have to manually select one protein to search for functional linkages.

    For queries submitted directly from the main page, only default settings are used. However, there are forms for advanced searches where parameters can be fine tuned as you see fit.

    Inferred linkages are presented in a table that also includes a link to visualize the network of functional interactions using Cytoscape (see the tutorial for further details). Each link can also be directly submitted to ProLinks, and each of the columns in the table can be sorted in ascending or descending order.

    Queries can be typed in the main page, or in the text box located in the top right corner of almost any page (e.g. this page or any advanced form) linked in the main page (see figure below and the homepage tutorial).

    Submit queries to ProLinks. Search boxes where you can write queries. Left. The search box in the homepage. Right. The search box at the top right corner in all other ProLinks related pages. Click on the boxes to see a full resolution version of the images.

    Find below a few examples you can use to query ProLinks. Click on Submit query! link to see the results.

    Description Query
    Submit a protein ID (Refseq). ID is unique therefore it is directly searched for functional linkages NP_414777    Submit query!
    Submit a gene name and its genome. Since there are several Escherichia genomes, there are several matches. proB Escherichia

    AND logic is used by defatult, but this alternative syntax can also be used:

    prob AND Escherichia
    +proB +Escherichia     		Submit query!
    Submit a protein name. If no genome is specified, the results will include all proteins in all genomes annotated with the same name. glutamylphosphate Submit query!
    Submit a protein function. Identify all proteins in E. coli K12 MG1655 annotated as participating in proline biosynthesis. The genome can be uniquely identified by typing the substrain (+MG1655) since there are currently 3 E. coli K12 strains in ProLinks. proline biosynthesis MG1655 Submit query!

  • Get links for multiple proteins you can run queries for multiple proteins by entering a list of public database protein identifiers, or by uploading a file with the identifiers (see valid formats). Results are returned by e-mail (click here to see the tutorial).

  • Find links shared between proteins: Submit several protein database identifiers to find out if they share any functional linkages. The list of identifiers can be directly entered or uploaded in a file (see valid formats). For more details visit the tutorial.

  • Find links connecting two proteins: Submit two protein database identifiers to identify if there are other proteins functionally linking them (a path in a network of functional linkages connecting the two proteins). Click here to see the tutorial.

  • Submit a protein sequence: If you only have the amino acid sequence of your protein of interest (in fasta format), you can submit it to ProLinks. A list of candidate homologous proteins will be returned, and you can select the protein that best represents your sequence to search for functional linkages (click here to see the tutorial).
