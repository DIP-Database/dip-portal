This number indicates each step in a pathway of linkages connecting the two query proteins.

If you would like to submit a given link back to ProLinks using only default parameters, click on the icon located to the right of the number. This is a shortcut for copying the ID of the protein and pasting it back on the homepage search box. If, on the other hand, you would like to resubmit an inferred linkage but modify the search parameters, click on the icon to open the respective advanced search form.
