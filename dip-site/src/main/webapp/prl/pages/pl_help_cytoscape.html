WARNING:
In order to be able to see the network, you must first install Cytoscape and the plugin MiSink in your computer. If they are not installed and running by the time you click on the icon, your browser will display an error similar to:

Firefox can't establish a connection to the server at 127.0.0.1:30403

We are working on providing a more informative message when this happens.




Once you click on the icon:


all inferred linkages in the current page are submitted to Cytoscape, via the MiSink plugin, in order to create a graphical and interactive display of the network of functional linkages.
