The index assigned to each blast-matching protein based on its statistical significance (as defined by the E-value).

If you would like to submit a given matched protein to ProLinks using only default parameters, click on the icon located to the right of the number. This is a shortcut for copying the ID of the protein and pasting it back on the homepage search box. If, on the other hand, you would like to resubmit a blast match but still be able to customize ProLinks thresholds, click on the icon to open the respective advanced search form.
