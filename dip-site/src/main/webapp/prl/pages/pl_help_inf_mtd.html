This colum presents the ProLinks method(s) responsible for inferring each functional linkage. See below for a brief description of each inference method:

Rosetta Stone (RS) uses gene fusion events in other organisms to infer functional relatedness.
Phylogenetic
Profiles (PP) 		uses the pattern of presence and absence of proteins across multiple genomes to detect functional linkages.
Gene Cluster (GC) uses likely membership to operons in order to infer functional linkages.
Gene Neighbor (GN) uses both the chromosomal proximity of genes and the conservation of such proximity across multiple organisms to infer linkages.
