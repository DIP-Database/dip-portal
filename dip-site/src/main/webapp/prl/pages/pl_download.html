Download ProLinks

  Currently ProLinks can be downloaded as tab-delimited text files compressed with bzip2. Files are organized in four columns: 1) RefSeq of the reference protein; 2) RefSeq of the inferred functionally linked protein; 3) The method(s) inferring the link between both proteins. If a link is inferred by more than one method, a list of comma-separated methods is given; and 4) The confidence value. In order to limit the size of the files, only inferences with confidence value ≥ 0.4 were included:
