Advanced searches in ProLinks

Protein keywords: Search ProLinks by submitting indentifiers, names, genomes, or any combination thereof, that can be used to identify a protein (click here for a list of suported database IDs). Complex queries including logical operators and wildcards are supported (see the tutorial for more details).
Multiple protein IDs: Search ProLinks in batch mode by submitting multiple protein IDs or by uploading a file with the IDs. Results are returned via e-mail.
Functional linkages shared among proteins: Find the set of functional linkages that is common to several query proteins. Protein IDs must be provided either directly or by uploading a file with the IDs. Results are returned via e-mail.
Paths of functional linkages connecting two proteins: Find the set of functional linkages that connects two proteins. For example, for two query proteins A and E, ProLinks could find that they are connected by proteins B, C, and D, that is:
A -- B -- C -- D -- E
Submit a protein sequence: A user-suplied input protein sequence is blasted against all procaryotic proteins in ProLinks, the most significant hits are then returned to the user sorted by statistical significance. Finally the user must select any of the returned hits and submit it to ProLinks to search for functional linkages.
