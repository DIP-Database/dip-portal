The values entered in these fields indicate the worst acceptable thresholds for each inference method implemented in ProLinks. Since the scores were calculated using different methods, they have different interpretations (see below). If you want to prevent this score from being used, for example, and rely only on the ProLinks score, enter any number greater than one to turn it off. For more details on how these scores were calculated, see the following descriptions and our main publication (2004, Genome Biol. 5(5):R35):

  • Phylogenetic Profiles (PP): In short, this score is a log ratio between two p-values. The more negative the value the more significant. For specific details in the calculation of this score, please see the report by Cokus et al (2007, BMC Bioinformatics 8 (Suppl 4):S7). ProLinks accepts values ≤-10.

  • Gene Neighbor (GN): This score is a p-value estimated by combining multiple probabilities, which results in very small values. ProLinks accepts values ≤1e-15.

  • Gene Cluster (GC): This score is a p-value. The most significant inferences are observed when the p-value is ≤0.01. ProLinks accepts values ≤0.02.

  • Rosetta Stone (RS): This score is a p-value. Similarly to the Gene Cluster method, the most significant inferences are observed when the p-value is ≤0.01. ProLins accepts any value ≤0.1.
