Organization of the website

In this tutorial you will attain a general understanding of the structure and organization of the entire ProLinks website, but here we will focus on the main homepage (see Figure 1). You will also be able to write queries to extract functional linkages by submitting a protein identifier, say, an NCBI RefSeq accession number (e.g. NP_415120, NP_417808, etc.) or by submtting gene/protein names, functions and/or genomes associated with your protein of interest.

Basic query

All you need to run a search is to know a public database identifier (click here for a list of supported IDs) or function-related keyword(s) that can be used to unambiguously indentify your protein of interest. For simplicity let's submit the RefSeq of a protein. The search will be carried out using default parameters.

There are two ways to run the search. First, use the text box SEARCH ProLinks for functional linkages (Figure 8 left, and Figure 1). Second, use the search box at the top right corner of most other ProLinks-related pages (see Figure 8 right, Figure 3, Figure 4, Figure 5, and Figure 6).

Figure 8. Basic query to ProLinks. Left. Using the search box in the homepage (Figure 1). Right. Using the search box in the top right corner in all other ProLinks related pages.

If you enter the RefSeq NP_414777 in any of the 2 search boxes shown in Figure 8, ProLinks will search for functional linkages and return the results in a table with additional relevant information as shown in Figure 9.

Figure 9. Inferred functional linkages. Results are presented in a table with 7 (or 8) columns. Most column names are self explanatory. Clicking on any column name opens a pop-up window with a description of the values shown in that particular column. The first column includes 2 icons for each linkage. The icon allows direct submission of the linked protein to ProLinks using default parameters. The icon gives the user the option to submit the linked protein as and advanced search, where all available search parameters can be modified. All colums with up/down arrows to the right of the name indicate that the whole table can be sorted increasingly or decreasingly, respectively, by the values in that column. (click on the figure to see the full size image).

Notice how the column "Inference Method" for the first link, proA, indicates that this particular link was inferred by 3 different methods.

The icon on the top right corner of the results table provides a means to generate an interactive graph of the network of inferred linkages using the software Cytoscape. Both Cytoscape and the plugin MiSink need to be installed and running at the time you click on the icon. Given that only six linkages were found in this example, it may not be worth to plot the network. However, this topic is covered in more detail in a different tutorial.
