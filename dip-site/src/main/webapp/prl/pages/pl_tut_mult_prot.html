Submit multiple protein identifiers

This tutorial is divided in 3 sections:
  • The input form: General description of the input form.

  • Submit a list of identifiers: Query ProLinks by entering a list of protein IDs.

  • Submit a file with the identifiers: Query ProLinks by uploading a file file with the protein IDs.

The input form

The input form to submit multiple proteins to ProLinks is presented in Figure 1a. The user must enter an e-mail (resuts are returned by e-mail only). The user can either copy/paste a list of protein identifiers or upload a local file with the IDs. Click here to see the current list of supported identifiers. The Search Parameters tab (see Figure 1b) accepts input values for Confidence value, Network Distance, and Indentify genes with.

IMPORTANT: if both a list of IDs and a file with IDs are provided, the entered list of IDs takes precedence and the file is ignored.


Figure 1a. Input form. The user must provide a valid e-mail address, and either enter the list of IDs or upload a file with the IDs. There are a number of valid identifier which you can see here. By clicking on any question mark, a popup window will open with an explanation of the corresponding topic. Figure 1b. The search Parameters are explained in the previous tutorial.


Submit a list of identifiers

If you click on the link: (see valid format) a window will open with an explanation of the accepted formats for the list you submit. The window shows 6 Locus tags in three different formats as an example. Copy/paste any ot the three lists of IDs into the correspondig text box and type in your e-mail. (see Figure 2).

Figure 2. Submit a list of protein IDs. The figure illustrates how the input form looks when it is filled with data.


Let's run this example using default parameters. When you click on the Search ProLinks button, ProLinks will display a page indicating that your query has been submitted to the cluster. Depending on the number of IDs you submit it will take longer for ProLinks to process your query. When it is done, you will get an e-mail with the a link where you can see and download the file with your inferences. This file includes a header with an description of each column. Figure 3A presents the results when we select the option Gene name from the output format menu Identify links with, and Figure 3B presents the results when we select the option NCBI RefSeq.

Submit a file with the identifiers

Instead of entering a list or copy/pase a list of identifiers, you can submit the identifiers in a text file. Then click on the Browse button of the input field File of Figure 1 and select the file you created. Once you select your file click on the Search ProLinks button, the results will be identical to those shown in Figure 3.
