The E-value or Expect value is the number of BLAST hits expected to be found by chance alone (false positives) when comparing your query sequence against all proteins contained in ProLinks. Click here to see a more detailed description of this parameter at the official BLAST site.

The closer this number to 0.0 the more significant. If the calculated E-value is very small (e.g. less than 1e-300), blast reports 0.0.
