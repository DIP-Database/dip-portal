The Cytoscape Export Panel provides means for exporting interactions into Cytoscape visualization program supplanted with the MiSink plugin.

Pressing one of the ALL or FLT button adds ALL or FILtered records listed in the table to the network currently open in a running copy of Cytoscape. Pressing the SEL button adds only these records that are currently selected within th table.

The Neighbor option specifies what interactions other then the set chosen above are also added: <1> adds interactions of the originally selected molecules, <1+> includes also interactions between the neighbors.
