Validation of the DIP links is performed in order to identify the most reliable subset of the database. This CORE subset can be used as a starting point for the analysis of the interaction network as well as a reference set in the development/evaluation of protein-protein interaction prediction methods.

The initial validation is performed by using two simple criteria:

  • a given link is considered reliable if it was identified in at least one non genome-wide experiment. In those cases the number of independent, small scale experiments (SMSC) is reported.
  • a given link is considered reliable if it was identified independently more than once. In those cases the number of all independent experiments (MLTI) is reported.

Additionally, each interaction is analyzed using the PVM approach which identifies interacting protein pairs with experimentally determined interactions between their respective paralogs. In those cases the number of paralogous interactions (PVMS) is reported (see Deane at al. for details).

The status of each link, as well as the criteria that contributed to its classification as a member of the CORE subset, are reported on the corresponding DIP LINK page. In addition the set of validated links from S. cerevisiae (DIP_CORE) can be downloaded as a list of validated protein pairs.
