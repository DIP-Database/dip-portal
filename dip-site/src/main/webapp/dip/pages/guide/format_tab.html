The species-specific subsets of the DIP database are provided also in a simple, tab-delimited format. The tab-separated columns contain: 

   - edge id
 
   - node A id 
   - node A SwissProt id
   - node A PIR id
   - node A GI code

   - node B id 
   - node B SwissProt id
   - node B PIR id
   - node B GI code
These files can be easily loaded into any standard spreadsheet application for further processing.
