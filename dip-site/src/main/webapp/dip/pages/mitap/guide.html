All the features provided by MiTap are available from the MiTap sidebar that can be activated by selecting MiTap option from the View->Sidebar menu.


A. MiTap tree provides access to information identified by MiTap as related to the currently displayed Web page. The Scan branch contains information directly extracted from the current page (or loaded from a file - see G), Expand contains elements added by clicking on Expand button (see D) and Filter - only elements selected by applying filtering criteria set with the Select tab.
B. Detailed information about the selected tree element is provided in the information panel at the bottom of the MiTap sidebar.
C. Links within the information panel can be used to directly access Web sites providing information relevant to the selected tree element.
D. Expand button adds interacting neighbours to all the proteins within the MiTap tree that were originally identified within the scanned page. The added proteins and interactions are added within the Expand branch.
E. Export buttons provide means for adding all (or selected) parts of the MiTap tree to the current network within Cytoscape (NOTE: Cytoscape must be running in order for the action to take effect).
F. Right-clicking on a tree element brings up a MiTap:Note dialog that can be used to locally annotate individual elements of the tree. Elements with attached annotation notes are highlighted in yellow. The annotation data is stored permanently within user's profile and thus persists between Firefox sessions.
G. File menu provides means for saving and reloading of the current MiTap tree.
