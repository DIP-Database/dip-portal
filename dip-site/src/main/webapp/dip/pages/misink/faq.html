MiSink Frequently Asked Questions:
1.

Q: I click on one of the links but nothing happens ?

A: Please, make sure Cytoscape is currently running and that it has been restarted after MiSink installation.

2.

Q: Can I load a local MIF 2.5 file into Cytoscape using MiSink ?

A: Yes. To load contents of a MIF 2.5 file into Cytoscape run the xmlTap program providing the file name as its argument:

misink/xmlTap my_mif_file.mif
NOTE:

1. Yes, you do have to start Cytoscape first.

2. The program is located in the misink subdirectory of the Cytoscape instalation directory. Thus the above command assumes it is run from the Cytoscape directory.

3.

Q: Can I load a local tab-delimited file into Cytoscape using MiSink ?

A: Yes. To load contents of a tab-delimited file into Cytoscape run the tabTap program providing the file name as its argument:

misink/tabTap my_tab_file.tab
NOTE:

1. Yes, you do have to start Cytoscape first.

2. The program is located in the misink subdirectory of the Cytoscape instalation directory. Thus the above command assumes it is run from the Cytoscape directory.
