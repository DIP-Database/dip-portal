MiSink is an add-on module (plugin) for Cytoscape - an open source platform for visualization and integration of biological interaction data. Installation of MiSink converts it into an interactive, graphical interface to DIP and, potentially other, other MiSink-enabled Web sites. This is possible, from the end-user perspective, by implementing, within MiSink, three additional operations:

  1. adding to the current Cytoscape network nodes and edges 'advertised' by a Web site through and icons.

  2. visiting sites that provide annotation of the imported nodes/edges

  3. finding information about the neighbouring nodes

In addition, MiSink can be used to import into Cytoscape data from tab-delimited or XML files. Data in practically arbitrary format can be imported thanks to strightforward configuration based on simple, yet powerful, XPath language.

A. The 'cDIP' icons available on some of the DIP Web pages can be used to load additional information into the current Cytoscape network. Depending on the context, interactions (green for core, blue for all) or just nodes can be imported.

B. Clicking on one of the 'cDIP' icons resulted in loading of the additional interactions into the current network.

C. Choosing 'UniProt' or 'RefSeq' options from the pulldown menu brings up corresponding page on, respectively, UniProt or RefSeq site.

D. An additional menu (accessible through the right mouse click) is available after installation of the MiSink Plugin for nodes uploaded from DIP web site. Choosing 'DIP' option there loads corresponding DIP web page into the default web browser.
