The following utilities can be used to create, view or validate files of the Molecular Interaction Format (MIF v 2.5). For more information on MIF files, check out the MIF website hosted by the Proteomics Standards Initiative (PSI), or this publication:

Broadening the Horizon - Level 2.5 of the HUPO-PSI Format for Molecular Interactions
Kerrien et al., BMC Biology 2007, 5:44.
MIF Maker: FormIt
The Web-based FormIt tool is an easy, intuitive utility for producing MIF 2.5 files. Basic information is entered via a series of Web forms, with controlled vocabulary terms supplied by pull-down menus.

MIF Viewer
This simple utility displays the contents of an input MIF 2.5.3 file.
Note: The MIF Viewer should cope well with files produced by IMEx Consortium members and those generated using the FormIt tool. However, due to the flexibility of the MIF format, it may fail when attempting to display files generated by other means.

MIF Syntax Checker
This utility attempts to validate the syntax of a MIF file. Please note that it tests formal agreement with the MIF 2.5.3 syntax specification, but makes no attempt to validate the semantics of the file. It is thus possible that a file that passes the validation here still contains semantic errors.
File

MIF Validator
The PSI-endorsed MIF Validator, provided by the IntAct database, checks formal syntax and also performs semantic validation.
