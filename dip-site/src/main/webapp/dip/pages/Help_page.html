ProLinks Help

The ProLinks website is organized in three sections: (1) how to make a query; (2) how to download all ProLinks inferences; and (3) finding help.

How to make a query
There are five different ways, in which ProLinks can be queried so far:

  • Submit a public database identifier: currently only NCBI RefSeq, NCBI GIs, and Swissprot IDs are accepted. Inferred linkages are presented in a table that also includes a link to see the network of functional interactions using Cytoscape ( click here to see the tutorials).

  • Submit a protein attribute: currently gene name, protein name, and a description of the function of the proteins are accepted. If there are multiple matches the system will request the user to select one. Inferred linkages are presented in a table that also includes a link to see the network of functional interactions using Cytoscape ( click here to see the tutorials).

  • Get links for multiple proteins you can run queries for multiple proteins by entering a list of public database protein identifiers, or by uploading a file with the identifiers. Results are returned by e-mail ( click here to see the tutorials).

  • Find links shared between proteins: Submit several protein database identifiers to find out if they share any functional linkages. The list of identifiers can be directly entered or uploaded in a file. (This option is still under development and will be available soon!)

  • Find links connecting two proteins: Submit two protein database identifiers to identify if there are other proteins functionally linking them (a path in a network of functional linkages connecting the two proteins. (This option is still under development and will be available soon!)
